# Styling in React

## Common Options
- Tailwind CSS
- StyleX
- Bootstrap
- Material UI (MUI)
- Ant Design (Ant UI)
- CSS / SCSS

## How to Choose
- **Speed**: Utility frameworks like Tailwind are fast to build with.
- **Design System**: MUI/Ant offer ready components.
- **Customization**: CSS/SCSS gives deep control.
- **Consistency**: Component libraries help teams scale UI.

## Best Practices
- Keep styles predictable and consistent.
- Prefer design tokens/theming for scalable systems.
- Avoid style duplication across components.

## Interview Tip
Explain trade-offs instead of saying one tool is always best.
